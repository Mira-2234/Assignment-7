# 🌿 KeenKeeper — Keep Your Friendships Alive

## 📌 Project Overview

**KeenKeeper** is a responsive React-based web application designed to help users maintain and nurture meaningful friendships. It allows users to track interactions, set communication goals, and stay connected with their friends through an intuitive and visually appealing interface.

---

## 🛠️ Technologies Used

* ⚛️ React.js
* 🔁 React Router DOM
* 🎨 Tailwind CSS
* 📊 Recharts
* 🔔 React Toastify (for notifications)

---

## ✨ Key Features

### 👥 Friend Management

* View all friends in a responsive grid layout
* Each friend card shows photo, name, tags, and contact status
* Click to view detailed friend profile

### 📅 Interaction Tracking (Timeline)

* Log interactions (Call, Text, Video)
* Automatically records date and action
* Timeline page shows all history with icons

### 📊 Friendship Analytics

* Visual Pie Chart showing interaction types
* Helps analyze communication patterns

### 🎯 Relationship Goals

* Set and track communication goals
* View days since last contact and next due date

### 🔔 Smart Notifications

* Toast notifications on interactions (Call/Text/Video)

---

## 📄 Pages Included

* 🏠 Home Page
* 👤 Friend Details Page
* 📜 Timeline Page
* 📊 Stats (Analytics) Page
* ❌ 404 Not Found Page

---

## 📱 Responsive Design

* Fully responsive for:

  * 📱 Mobile
  * 💻 Desktop
  * 📟 Tablet

---

## 📂 Data Source

* Friend data is stored in a local JSON file
* Includes realistic profile information such as:

  * Name, Email
  * Days since contact
  * Status (on-track / almost due / overdue)
  * Tags and bio
